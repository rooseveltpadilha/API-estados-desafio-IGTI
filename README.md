

## ESTADOS API

API feita com express + cors. Criada para resolver uma lista de problemas de uma atividade do IGTI.

![lista de atividades](https://i.ibb.co/KqzSz0J/image.png)

Para startar:

```
npm install
```
Com este comando você irá inicializar o servidor:
```
npm run dev
```
